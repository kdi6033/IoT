### Calibration Factor
* calibration factor값을 지정해서 scale값을 세팅해줌
* 사용자에 맞는 calibration factor지정을 위해 변경가능하도록 프로그래밍
* 시리얼모니터에 1,2,3,4  a,s,d,f 입력 시마다 변경 가능  
* Loadcell의 출력값이 -0 ~ 0이 나올때 까지 보정
